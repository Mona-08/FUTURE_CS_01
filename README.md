# 🔐 Vulnerability Assessment Report

## 📌 Website Tested

http://testphp.vulnweb.com

## 🎯 Scope

* Read-only testing only
* No exploitation performed

## 🛠️ Tools Used

* Nmap
* OWASP ZAP (Passive Scan)
* Browser DevTools

## 🔍 Key Findings

* Missing security headers
* Overly permissive CORS policy
* Information disclosure (server details)

## ⚠️ Risk Levels

* Medium: Security misconfigurations
* Low: Information exposure

## 🧾 Description

This project demonstrates a basic vulnerability assessment performed on a public website.
The goal is to identify common security issues and provide remediation steps in a business-friendly manner.

## 📂 Files Included

* report.pdf → Final professional report
* screenshots/ → Evidence
* nmap-results.txt → Scan output
